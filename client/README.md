## Overview
This project is a modern React web application that searches and displays photos using the **public Flickr photo feed**.

The app implements:
- a search UI (tag-based)
- responsive layout with Tailwind CSS
- loading indicators (global + per image)
- client-side pagination
- basic rate-limit protection (user-friendly message)

## Tech stack
- React (Vite)
- JavaScript
- Tailwind CSS
- Fetch API
- Vite proxy (to avoid CORS issues)
